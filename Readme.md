#### Installation for running the code:

1. Install Go: https://golang.org/doc/install


```
$> git clone: https://github.com/motiooon/Affirm-Go.git
$> cd Affirm-Go
$> go run Affirm
```

After that the 2 CSV files will be added to the root folder of the program.
